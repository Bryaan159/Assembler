*Que representa un Jacobinano*
Representa una matriz que contiene las derivadas parciales de una transformacion en un sistema de coordenadas es decir de (x,y)
hacia el (e,n)

---------------------------------------------------------------------------------
Para qué se utiliza el Jacobiano en el MEF

Se utiliza basicamente para transformar un sistema fisico a un sistema paramétrico de referencia en pocas palabras es para poder
usar las funciones de forma y se encuentre derivadas por dEdN
